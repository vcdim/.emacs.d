# Installation Guide

This is my personal `.emacs.d` configuration. To install:

1. Install `emacs-plus-app` by

   ```sh
   brew install emacs-plus-app
   ````

2. Clone the repository to home folder (`~`) by

   ```sh
   git clone https://github.com/vcdim/.emacs.d.git ~/.emacs.d/
   ```

3. Install libvterm and cmake
   ```sh
   brew install cmake
   brew install libvterm
   ```

4. Install fonts inside emacs
   ```sh
   M-x all-the-icons-install-fonts
   M-x nerd-font-install-fonts
   ```
